Roman Name Attestations

This map explores personal names recorded in Roman inscriptions across the imperial provinces. Data is drawn from the Epigraphic Database Clauss-Slaby (EDCS), which contains over 500,000 Latin inscriptions.

Names were extracted using automated Named Entity Recognition (NER) and grouped into prosopographical clusters — sets of inscriptions that likely refer to the same individual. Each marker on the map represents one or more inscriptions from a single find-spot.

How to use:

  • Select a province to focus the map
  • Search by name (praenomen, nomen, or cognomen)
  • Click a marker to see the people named in that inscription
  • Use filters to explore by gender, confidence level, or social category
  • Follow "See all N attestations" to find all known appearances of a name
Data: EDCS, LIRE v1.2, CIL, EDH, Trismegistos, Ubi Erat Lupa. Map tiles: DARE.
